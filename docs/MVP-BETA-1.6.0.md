# MVP Beta 1.6.0

## Objetivo

Esta version mantiene la estructura actual de WordPress y deja preparada la base para una beta operativa de Captacion.app.

## Fuente

- `stable-1.6.0/captacion-app`

## Que conserva

- Login y panel privado.
- Publicaciones y demandas.
- Matching automatico.
- Notificaciones.
- Consentimiento y trazabilidad.
- Medios, imagenes y video.
- Catastro y documentos legales.

## Que prioriza

- Registro gratuito.
- Publicaciones ilimitadas.
- Coincidencia con activacion de contacto tras pago de servicio.
- Panel personal por usuario.
- Programa de referidos basado en valor real.

## Uso previsto

Esta copia sirve como base de despliegue para la beta sin sustituir la version actual hasta que se confirme la publicacion.
