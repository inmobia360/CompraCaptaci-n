CAPTACION.APP — PAQUETE MULTIPÁGINA PARA HOSTINGER
=================================================

1. Haz una copia de seguridad de la carpeta public_html del dominio.
2. Descomprime el contenido de este ZIP.
3. Sube TODO el contenido de la carpeta captacion_app_hostinger directamente dentro de public_html.
4. Verifica que index.html quede directamente dentro de public_html.
5. Comprueba en el navegador: inicio, menú móvil, páginas internas, filtros, formularios y sitemap.xml.
6. En Google Search Console, añade la propiedad del dominio y envía: https://captacion.app/sitemap.xml

IMPORTANTE
- La web entregada es una base estática multipágina optimizada para indexación.
- Los formularios simulan el guardado local; para recibir registros reales necesitas conectarlos a un backend, CRM o automatización.
- El Área privada es demostrativa y está marcada como noindex.
- Completa y revisa jurídicamente aviso legal, privacidad y cookies antes del lanzamiento comercial.
- Si utilizas otro dominio, sustituye https://captacion.app en las etiquetas canonical, sitemap.xml y robots.txt.

MEJORAS FUNCIONALES INCORPORADAS
- Publicación de necesidades de compradores activos con filtros geográficos.
- Dashboard de demanda por comunidad, provincia y municipio.
- Vista de bloques y lista.
- Marketplace dinámico con ordenación y solicitud de acceso.
- Formulario completo para ofrecer captaciones con previsualización.
- Carga local de imágenes optimizadas (máximo recomendado: 1,2 MB).
- Área privada demostrativa con KPIs y baja de captaciones.
- Calculadora de honorarios y descargas orientativas.
- Funciones de IA preparadas para conectarse a un endpoint seguro de backend.

La versión estática utiliza localStorage para demostración. Para un entorno multiusuario real debes conectar autenticación, base de datos, almacenamiento de archivos, firma y backend seguro.
